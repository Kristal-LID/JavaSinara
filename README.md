какие параметры есть в SpringBootApplication?

В аннотации @SpringBootApplication есть несколько параметров, которые можно настраивать. Она является составной аннотацией, объединяющей в себе три другие:

__@Configuration:__ Указывает, что класс может быть использован для определения бинов.
__@EnableAutoConfiguration:__ Включает автоматическую настройку Spring Boot.
__@ComponentScan:__ Указывает Spring сканировать пакеты на наличие компонентов, конфигураций и сервисов, чтобы автоматически обнаруживать и регистрировать их.


## Параметры @SpringBootApplication

1.	_scanBasePackages_
Указывает пакеты, в которых Spring Boot будет искать компоненты, сервисы и конфигурации.
2.	_scanBasePackageClasses_
Задает классы, пакеты которых будут использованы как базовые для сканирования.
3. 	_exclude_
Исключает определенные классы из автоконфигурации.
4. 	_excludeName_
То же, что exclude, но используется, если классы отсутствуют в classpath
